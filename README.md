Sign-Up Page Replication Assignment
Overview
This project is a pixel-perfect implementation of a sign-up page UI mockup using only HTML and CSS. 
The assignment focuses on translating a visual design into functional, structured, and well-styled code without any JavaScript functionality.

Objective
To practice translating a UI/UX mockup into code by focusing on:

Structure: Semantic HTML markup

Layout: CSS-based positioning and responsiveness

Typography: Font styles, sizes, and spacing

Visual Accuracy: Matching the provided mockup as closely as possible
 Features
Clean and modern sign-up form

Responsive design principles

Semantic HTML5 elements

CSS custom properties for maintainability

No external frameworks or libraries

No JavaScript (HTML & CSS only)
Technologies Used
HTML5: Semantic markup

CSS3: Flexbox/Grid, custom properties, responsive units
